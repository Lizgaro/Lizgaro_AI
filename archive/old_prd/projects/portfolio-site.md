# Project — Portfolio Site / личная упаковка

## Статус

Приоритетный проект. Нужен как первый публичный актив и точка продаж.

## Главная цель

Создать русскоязычный сайт-портфолио Артёма Лизгаро, который:

- быстро объясняет, кто он;
- показывает путь и мышление;
- продаёт консультации, сайты, боты, маркетинг и рост;
- показывает кейсы и направления;
- собирает заявки;
- становится базовой ссылкой для Telegram, X, Instagram/Threads, VK/Max и партнёрств.

## Позиционирование v1

```txt
Артём Лизгаро
AI-маркетолог, builder и growth-оператор.
Создаю сайты, боты, воронки и AI-системы, которые помогают бизнесам и проектам расти.
```

## Более жёсткий hero-вариант

```txt
От хаоса идей к сайтам, ботам, трафику и продажам.
```

## Более продуктовый hero-вариант

```txt
Собираю AI-продукты, воронки и аудитории вокруг сильных идей.
```

## Более консультационный hero-вариант

```txt
Помогаю фаундерам, экспертам и командам упаковывать идеи, строить воронки и запускать AI-системы.
```

## Главная аудитория сайта

1. Фаундеры SaaS/AI-продуктов.
2. Инфлюенсеры и владельцы аудиторий.
3. Эксперты, которым нужен сайт, бот, блог, воронка или продвижение.
4. Малый/средний бизнес, которому нужна AI/маркетинг-система.
5. Будущие партнёры / кофаундеры.
6. Подписчики, которым интересны AI, саморазвитие, продукты и путь.

## Визуальное направление

Итоговый стиль:

```txt
Dark AI Operator + Neon Lime + Editorial Portfolio
```

Что взять из референсов:

- Devsync: тёмный техно-вайб, огромный hero, skill-блок, проекты-сетка, сервисы, процесс, мощный финальный CTA.
- Portavia: яркая типографика, аккордеоны услуг, большие карточки проектов, отзывы/FAQ/форма, лаймовый акцент.
- Billie: крупный editorial hero, асимметричная сетка, ощущение премиальности, много воздуха, минимальная навигация.

Что не копировать:

- чужие фото;
- тексты;
- названия;
- логотипы;
- точную композицию;
- made in Framer бейдж;
- чужие выдуманные отзывы;
- фейковые клиенты.

## Типографика v1

- Большие заголовки: сжатые/жёсткие, uppercase, высокий контраст.
- Основной текст: простой, читаемый, без декоративности.
- Подписи: моноширинные или техно-стиль, маленькие капсы.

Пример:

```txt
Display: Space Grotesk / Oswald / system condensed fallback
Body: Inter / system sans-serif
Mono labels: JetBrains Mono / ui-monospace
```

## MVP сайта

Первый релиз:

1. Header.
2. Hero.
3. About.
4. What I do.
5. Skills.
6. Cases.
7. Offers.
8. Philosophy.
9. CTA/contact.
10. Footer.

Не делать в первом релизе:

- полноценный блог;
- CMS;
- сложные 3D;
- личный кабинет;
- 20 страниц;
- сложные анимации;
- фейковые отзывы.

## Done when

Сайт v1 готов, когда:

- на русском;
- понятно, кто такой Артём;
- понятно, чем он полезен;
- есть CTA;
- есть 4–5 кейсов/направлений;
- есть услуги;
- есть адаптив;
- сайт можно отправить потенциальному клиенту/партнёру;
- он не копирует референсы один-в-один;
- его можно развивать дальше.
