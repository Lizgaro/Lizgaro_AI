# Документация сайта Артёма Лизгаро

## Главный PRD

Единственный источник правды для сайта:

`docs/MASTER_PRD_Artem_Lizgaro_final_site_social.md`

Все решения по структуре, текстам, CTA, SEO, соцсетям и визуальной логике сверяются с этим файлом.

## Устаревшие документы

Старые PRD, брифы и промежуточные документы перенесены в:

`archive/old_prd/`

Они оставлены только как история. Использовать их как контекст, источник формулировок или аргумент против master PRD запрещено.

## Что нельзя возвращать

- подкастеров как целевую аудиторию или оффер;
- Life OS как клиентскую услугу;
- CTA `Хочу собрать свой AI-мозг`;
- формулировки про `личный AI-контекст`;
- `контент-систему под ключ` как отдельный оффер;
- перегруженный AI/agency-тон;
- старую логику “сайт, бот и AI” как отдельные услуги.

## Как работать дальше

1. Сначала открыть `docs/MASTER_PRD_Artem_Lizgaro_final_site_social.md`.
2. Работать только в папке `Lizgaro_site`.
3. Менять сайт по карте лендинга из master PRD.
4. После крупных этапов обновлять `docs/agent_checkpoint.md`.
5. Перед завершением запускать:

```powershell
npm run lint
npm run build
```

Команды одинаково подходят для PowerShell, Command Prompt и Warp после перехода в папку:

```powershell
cd C:\LA\life-os-v0.3\Lizgaro_site
```
